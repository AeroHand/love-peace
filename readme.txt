游戏采用love2d引擎开发


LOVE文件夹内封装了love引擎
LOVE文件夹中的I文件夹为游戏代码本体
I文件夹中的document.txt为代码文档 同时代码中我也会尽量做详细注释

test版本号.zip一般为打包好的exe可执行游戏文件

游戏测试方法：
下载整个repo 在LOVE中直接把I文件夹拖到love.exe中即可
